# A 股研究手账 v1.0.0

第一版可用版本，目标是作为个人 A 股学习、复盘、判断和打卡的日记型工具。

## 已完成

- 六个工作表：总览、行情大屏、公司研究、反思判断、每周计划、打卡本
- 暗色界面、粒子动态、向上积累视觉、像素打卡奖励
- 公司研究基础框架：公司背景、融资逻辑、靠什么赚钱、行业地位、上涨逻辑、风险、一句话理解
- 自定义研究问题、研究问题池、资料来源、研究完成度、公司报告生成和复制
- 判断复盘：观察、可以买、不买、放弃，包含证据、疑点、风险、买入条件、放弃条件、下次复盘日期
- 判断完整度评分、买入前检查表、待复盘判断提醒
- 行情大屏：指数、板块、市场节奏、重点观察股票、市场笔记
- 每周任务打卡、本周摘要、周报归档、开启新周
- 周一到周日生活日记、每日市场复盘、每日判断反思、明日行动
- JSON 导入导出、Markdown 导出
- 本地 `/api/state` 保存，Cloudflare Pages + D1 Function 预留
- `/api/health` 健康检查
- 状态版本号、导入校验、接口非对象状态拦截
- 桌面和 390px 手机宽度响应式检查通过

## 本地运行

```bash
npm run dev
```

默认地址：

```text
http://127.0.0.1:4176
```

当前开发时如果 4176 被占用，可以用：

```bash
PORT=4178 npm run dev
```

## 最终检查

```bash
npm run qa
```

该命令会执行语法检查和本地接口 smoke 测试。

## 部署

部署到 Cloudflare Pages + D1 前，按 [DEPLOYMENT.md](./DEPLOYMENT.md) 完成 D1 创建、绑定和表结构初始化。
